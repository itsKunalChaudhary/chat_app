## Backend Setup
- Navigate to backend folder:

```bash
cd backend
```

- Install dependencies:

```bash
npm install
```
- Create a .env file:

```ini
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```

- Start the server:

```bash
npm start
```
